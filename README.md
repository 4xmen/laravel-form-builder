# laravel-form-builder
The laravel form builder code generator

## how to use it

It so easy, Let's go (O_x) 

### download project
git clone or download: 

```shell script
git clone https://gitgo.ir/4xmen/laravel-form-builder.git
```

### install dependencies:

```shell script
  npm install
 ```

## how run it
open `index.html`
